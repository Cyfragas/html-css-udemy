O  que é o DOM

Document Object Model

- API que permite o acesso via JavaScript aos elementos da página.

- Todos os elementos HTMl ficam organizados em um document, uma representação análoga a uma árvore onde os elementos pais encapsulam os elementos filhos traçando trajetos bem específicos para alcançar cada um dos elementos que estrturam a página.

- Árvores de elementos do DOM

- Cada elemento contido no DOM é também chamado de nó ou node em inglês.
